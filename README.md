# deadline
Task management tool with a twist

## Angular2
 
### Prerequisites

The project has dependencies that require Node 6.9.0 or higher, together with NPM 3 or higher.

### Install Angular-CLI
```shell
npm install -g @angular/cli
```

### Usage 
```shell
cd Angular2Test
run ng serve
```
